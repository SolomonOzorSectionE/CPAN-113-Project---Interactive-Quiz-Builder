What This Website Does

This website is designed for students who want to build practice quizzes, test their knowledge, and get instant feedback on their performance.

Users can create quiz questions, choose correct answers, attempt quizzes, and view their final scores.

Main Features

Create Quiz

Add quiz names, questions, multiple options, and correct answers.

All Quizzes

See all created quizzes in one place and review saved questions.

Answer Bank

View stored quiz questions together with their correct answers.

Quiz Results

Check total questions, stored results, and final score summaries.
